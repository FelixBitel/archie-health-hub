# Archie Health Dashboard - Анализ проекта

## Описание
AI-powered система мониторинга здоровья питомца с интеграцией Qwen3-8B

## Цели
- Современный веб-дашборд для health tracking
- AI-ассистент для рекомендаций (Qwen3)
- Mobile-first responsive design
- Деплой на GitHub Pages

## Технологический стек
- Frontend: HTML5, Tailwind CSS, Vanilla JS (ES6+)
- AI: Jan.ai API (http://localhost:1337) - Qwen3-8B
- Data: LocalStorage + JSON
- Charts: Chart.js
- Deploy: GitHub Pages

## MVP Features (Phase 1)
1. Health Stats Dashboard (вес, рост, BMI)
2. AI Chat Assistant (Qwen3)
3. Activity Tracker (прогулки, игры)
4. Basic Analytics (графики)

## Future Features (Phase 2-3)
- Medication reminders
- Vet appointments calendar
- Photo gallery
- Browser notifications
- Export data (PDF/CSV)

## Связь с archie-automation
- Backup скрипты копируют данные
- Deploy скрипты пушат в GitHub
- Логи сохраняются для отладки
